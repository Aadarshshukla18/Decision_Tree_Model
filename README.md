# Decision_Tree_Model# 🌳 Student Performance Prediction using Decision Tree

This project uses a **Decision Tree Classifier** to predict whether a student will **pass or fail** based on multiple factors like study hours, attendance, sleep, and previous academic performance.

---

## 📌 Project Overview

The objective of this project is to analyze student behavior and build a machine learning model that can predict academic success using classification techniques.

---

## 📊 Dataset Features

- Hours_Studied → Number of hours a student studies  
- Attendance → Class attendance percentage  
- Sleep_Hours → Daily sleep duration  
- Previous_Score → Past academic performance  

### 🎯 Target Variable
- Result (0 = Fail, 1 = Pass)

---

## 🧠 Model Used

- Decision Tree Classifier

---

## ⚙️ Technologies Used

- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  
- Scikit-learn  

---

## 🔍 Steps Performed

1. Loaded and explored the dataset  
2. Visualized data using various plots (countplot, scatterplot, heatmap, etc.)  
3. Defined features (X) and target variable (y)  
4. Split data into training and testing sets  
5. Trained Decision Tree model  
6. Evaluated model using accuracy score  
7. Visualized the Decision Tree structure  

---

## 📈 Results

- The model achieved good accuracy on the dataset  
- Study hours and previous scores showed strong influence on results  
- Decision Tree successfully captured patterns in the data  

---

## 📊 Visualizations

- Count Plot (Pass vs Fail)  
- Scatter Plot (Feature relationships)  
- Pair Plot (Overall feature interaction)  
- Heatmap (Correlation between features)  
- Box Plot (Feature distribution by result)  

---

## 🧾 Conclusion

Students with higher study hours and better previous scores are more likely to pass. Attendance and sleep also contribute to performance. The Decision Tree model effectively identifies patterns but may overfit if not controlled properly.
